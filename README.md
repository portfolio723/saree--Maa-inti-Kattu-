# Maa Inti Kattu

This is a Next.js e-commerce application for "Maa Inti Kattu", a store for premium traditional goods.

## Features

- Product Browsing with filters and search
- Secure Checkout
- Order Tracking
- User Authentication
- Product Management (Admin)
- Customer Reviews and Ratings

Built with Next.js, TypeScript, Tailwind CSS, and ShadCN UI.
