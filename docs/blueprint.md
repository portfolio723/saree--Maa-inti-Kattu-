# **App Name**: Maa Inti Kattu

## Core Features:

- Product Browsing: Browse products with filtering, sorting, and search capabilities.
- Secure Checkout: Complete purchase using Razorpay with multiple payment methods.
- Order Tracking: Track order status in real-time with delivery timeline visualization.
- User Authentication: Allow users to register, log in, and manage profiles using Supabase.
- Product Management: Admin interface for managing products, categories, and inventory.
- Customer Reviews: Allow users to write text-based reviews and ratings for each product. Also provide the ability for other users to upvote and downvote each rating.

## Style Guidelines:

- Primary color: Deep maroon (#800000), evoking premium traditional Telugu culture.
- Background color: Desaturated maroon (#F0E0E0), provides a soft and elegant backdrop.
- Accent color: Rich gold (#DAA520), used for CTAs and highlights, providing a touch of luxury.
- Headings and titles: 'Playfair Display' (serif) for an elegant, upscale look. Note: currently only Google Fonts are supported.
- Body and UI elements: 'Raleway' (sans-serif) for readability. Note: currently only Google Fonts are supported.
- Use gold-colored, detailed icons, in the style of traditional Indian motifs and patterns
- Subtle animations to highlight elements.
